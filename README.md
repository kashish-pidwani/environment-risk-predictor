# Environmental Risk Predictor 🌱

This project predicts environmental risk levels using machine learning.

## Features
- Data cleaning and preprocessing
- Risk classification using CO and NO2
- Random Forest model
- Visualization and analysis

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
  
## Model Performance
- Achieved 100% accuracy using Random Forest classifier
